artistic snapshots

Deb Barsoum Photography

Challenge

To create an artistic website for an artistic photographer. The website needed to present photos in accessible ways while preserving UX and non-photographic content. It would need to include bold headings and vibrant colors, as well as an interesting selection of photos throughout. It would also need to be responsive and fit any device, to allow anyone, anywhere, on any device to view it.

Role

I designed the website and provided content guidance for the photographer. I incorporated her photos throughout the site, as well as brief, important other content.

View code ↗

Results

The result is an artistic, visually appealing website that presents the photographer's work in a so-called 'darkroom.' It includes a custom, non-obtrusive cursor; big, bold headings; and strategically placed services and biography sections containing brief yet appealing content.
